# Vipere Web

A web application for managing and viewing Roblox group statistics. The app provides win loss records, members, and medal recipients. All is supported by an API outlined below.

---

## Features

### Public Features
- View all Roblox group members  
- See group's record (wins/losses)  
- Browse opponents each the group has fought and whether they won or lost  
- View medal recipients and their achievements
- All data automatically fetched from the backend  

### Admin Features
- Secure admin login  
- Insert new members, match results, and medals  
- Delete or update existing records  
- Backend-managed data validation

---

## Tech Stack
- **Frontend:** React.js
- **Backend:** Python
- **Database:** SQLITE
